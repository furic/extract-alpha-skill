# Privacy Policy

**Extract Alpha** is an open-source Claude Code plugin that processes images locally on your machine.

## Data Collection

This plugin does **not** collect, store, transmit, or share any personal data or user information.

## How It Works

- All image processing happens locally on your device
- No images or data are sent to any external server
- No analytics, tracking, or telemetry of any kind
- No account or registration required

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.

## Contact

If you have questions about this privacy policy, please open an issue at https://github.com/furic/extract-alpha/issues.

## Changes

Any changes to this policy will be reflected in this file and the repository's commit history.
